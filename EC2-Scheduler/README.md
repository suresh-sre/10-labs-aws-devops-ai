# EC2-Scheduler

This project is a scaffold for learning AWS, DevOps, and AI automation by building one module at a time.

## Goal
Schedule EC2 start/stop actions using rules and automation jobs.
## AWS Credentials
- `AWS_REGION` (optional if a default region is set in AWS config)
- `AWS_ACCESS_KEY_ID` (optional if using shared credentials/profile)
- `AWS_SECRET_ACCESS_KEY` (optional if using shared credentials/profile)
- `AWS_PROFILE` (optional; preferred when using named profiles)
## Suggested learning flow
1. Review the architecture notes.
2. Implement the configuration layer.
3. Add one service or workflow module.
4. Test it before expanding the scope.
